# Agent 初始化引导 — hermes-agent-onboarding-guide

系统入口文件：SKILL.md

**用途：** 新 Agent 标准化初始化 — 环境探测、用户画像、Skill 选择、模型分层
**加载：** skill_view(name='hermes-agent-onboarding-guide')
**相关：** vault-external-brain（外部记忆框架）

## 一句话

新 agent 最费 token？按 9 个 phase 走，避免重复摸索。

## 9 个 Phase

1. 环境探测
2. 用户画像
3. 工作区约定
4. Skill 选择
5. 模型分层（Flash / Pro / Vision）
6. 沟通契约
7. 记忆初始化（含 `[CORE]` 核心记忆规则）
8. Skill 创作纪律
9. 完成检查清单
